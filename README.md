## Requirements

- [Git](https://git-scm.com)
- [Node](https://nodejs.org/en/)
- [Ruby on Rails](https://rubyonrails.org/)

## Getting Started

<br>

```bash
cd LudiStock
bundle install
rails db:create
rails db:migrate
rails s
cd client
npm install
npm run dev
```
